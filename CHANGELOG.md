# Changelog

Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-04-20
### Adicionado
- Analisador de áudio com extração de BPM, Tom e Espectro de frequências.
- Classificador de gêneros e subgêneros focado em EDM (Electronic Dance Music).
- CLI para uso por linha de comando.
- Suporte a análise em lote.
- Opção para exportar resultados de pastas em formatos `csv` e `json`.
- Comando para comparação de faixas de áudio lado a lado.
- Interface Gráfica (GUI) para facilitar o uso da ferramenta.
- Visualização gráfica de espectro e dados com o parâmetro `--plot`.
